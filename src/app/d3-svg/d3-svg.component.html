¿Y si tengo los datos en un CSV / TSV / JSON?

D3.js permite importar archivos de datos directamente usando los métodos

Métodos
d3.csv(url, bindingFunction).then(callBack).catch(errFunction);
d3.tsv(url, bindingFunction).then(callBack).catch(errFunction);
d3.json(url, bindingFunction).then(callBack).catch(errFunction);
d3.dsv(url, bindingFunction).then(callBack).catch(errFunction);

La promesa siempre recibe el arreglo de datos finales y si existió algun error
